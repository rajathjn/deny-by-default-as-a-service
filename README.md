# Deny By Default as a Service

Created this fun project inspired from the below repo:
https://github.com/hotheadhacker/no-as-a-service

Ever needed a graceful way to say “no”?  
This tiny API returns random, generic, creative, and sometimes hilarious rejection reasons — perfectly suited for any scenario: personal, professional, student life, dev life, or just because.

Built for humans, excuses, and humor.

## API Usage

**Base URL**
```http
```

**Method:** `GET`
**Rate Limit:** `120 requests per minute per IP`

### Example Request
```http
GET /no
```

### Example Response
```text
"This feels like something Future Me would yell at Present Me for agreeing to."
```
